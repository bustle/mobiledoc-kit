Class PostNodeBuilder

The PostNodeBuilder is used to create new {@link Post} primitives, such as a MarkupSection, a CardSection, a Markup, etc. Every instance of an Editor has its own builder instance. The builder can be used inside an {@link Editor#run} callback to programmatically create new Post primitives to insert into the document. A PostNodeBuilder should be read from the Editor, not instantiated on its own.
